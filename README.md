# Python购物商城系统 python1050

## 项目概述

本项目是基于Python语言的购物商城系统，采用前后端分离的架构设计，前端使用Vue.js框架，后端使用Django框架，数据库采用MySQL。

## 技术栈

- 后端：Python + Django
- 前端：Vue.js
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户模块
- 商品模块
- 购物车模块
- 订单模块
- 支付模块
- 管理员模块

## 系统角色

- 管理员
- 用户

## 运行环境

- 前端：Node.js（版本高于14）
- 后端：Python 3.x

## 目录结构

```
项目目录
│
├── 前端代码
│   └── ...
└── 后端代码
    ├── apps
    │   ├── 用户模块
    │   ├── 商品模块
    │   ├── 购物车模块
    │   ├── 订单模块
    │   └── 支付模块
    ├── static
    └── ...
```

## 部署步骤

1. 准备Python运行环境
2. 安装依赖包
3. 配置数据库MySQL
4. 运行后端项目
5. 运行前端项目
